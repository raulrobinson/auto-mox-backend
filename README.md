# auto-mox-backend
Backend Nodejs for AutoMOX Control System

- **.env**

Environment variables:

```text
# --- Main ---
PORT=${port}

# --- Database ---
BD_HOST=${bd_host}
BD_USER=${bd_user}
BD_PASS=${bd_pass}
BD_BBDD=${bd_name}

# --- Access Control ---
APP_ID_MAIN=${app_id_main}
APP_ID_SUB=${app_id_sub}
```
